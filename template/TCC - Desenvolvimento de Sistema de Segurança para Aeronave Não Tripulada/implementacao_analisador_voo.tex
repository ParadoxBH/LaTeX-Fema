\newpage\subsection{ANALISADOR DE LOG DE VOO}
\p{O Analisador de Voo é um software 3d desenvolvido na Unity em C\# utilizando-se do Framework de Mapa Online chamado Infinity Code. O analisador tem como objetivo facilitar a leitura/visualização dos extenso arquivos de log em um ambiente 3d, onde o operador poderá assistir uma simulação do voo da aeronave, enquanto observa todos os dados da aeronave de acordo com a timeline de visualização.}
\insertimagesize{me}{analisadorVoo/principal}{Ambiente 3d no Analisador de Voo}{450px}
\p{A metodologia utilizada para realizar a leitura do histórico de voo gerado pelo sistema de segurança, é a mesma que o planejador de voo utiliza para receber os dados do sistema de segurança. Com a diferença que, em vez de receber os dados por uma conexão Socket, será recebido através da leitura de múltiplos arquivos gerado pelo sistema de segurança em seu Cartão de Memoria.}
